===== SLIDE 14 =====
POST-PROCESSING 
Group by processing 
Order by and distinct processing 
Evaluation for analytic function 




Group by processing ( qexec_groupby () ) 
      -  Aggregate function returns one result based on the group of rows.  (SUM, AVG ..) 
      - sort group by, hash group by 

Evaluation for analytic function ( Qexec_execute_analytics () ) 
      -  Analytic function calculates the aggregate value based on the result of rows.  (LAG, RANK ..) 

Order by and distinct processing ( qexec_orderby_distinct () ) 
      -  sort_listfile ()  에서  SORT_DUP  옵션으로 처리됨 . 
            SORT_ELIM_DUP : eliminate duplicate 
            SORT_DUP : allow duplicate
