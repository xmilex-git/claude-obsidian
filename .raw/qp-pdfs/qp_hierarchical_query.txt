===== SLIDE 2 =====
Hierarchical Query 
SELECT id,  mgrid 
FROM tree 
START WITH  mgrid  is null 
CONNECT BY PRIOR id= mgrid 

Hierarchical Query is used to obtain a set of data organized in a hierarchy 
The START WITH clause will filter the rows from which the hierarchy will start. 
The CONNECT BY condition is tested for a pair of rows. (parent-child relationship)
